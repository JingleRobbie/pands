## Decisions Made

- **`bind:indeterminate` not usable with `$derived`** ‚Äî Svelte 5 compile error (`constant_binding`); resolved with `bind:this` + `$effect` pattern instead
- **Unscheduled runs not deduplicated** ‚Äî merge only applies when `runDate` is non-null; unscheduled runs have no date to collide on so multiple are allowed
- **Lazy group creation** ‚Äî `scheduleGroup` defers creating the `production_run_groups` row until it knows at least one new run will be inserted; prevents orphan group rows when all lines merge
- **Route URLs unchanged** ‚Äî `/confirm` routes kept as-is; only visible UI text was renamed to "produce/produced"
