## Decisions Made

- **`$derived` expression vs function form** — use `$derived(expr)` for inline expressions, `$derived.by(() => { ... })` for multi-line function bodies. Plain `$derived(() => ...)` assigns the function object as the value — silent bug.
- **Print CSS global** — all `@media print` rules belong in `src/app.css`, not component `<style>` blocks, because Svelte scopes component styles with a hash that can't reach layout elements.
- **Packing slip stays per-line** — one row per WO line (not one row per individual roll), with Length added as a column.
