## Context Anchors

- `work_orders.customer_name` (VARCHAR) = raw name from import;
  `customer_display_name` = alias from LEFT JOIN on `customers.name`.
  Both exist and are used in WO-2 header logic.
- `redirect()` in SvelteKit throws internally — always place outside
  `try/catch`. CLAUDE.md documents this pattern.
- `$derived(expr)` not `$derived(() => expr)` — function form creates
  a function, not a reactive value. Confirmed live in this session.
- Shipment `?wo=` param threads through:
  `new/+page.server.js redirect` → `[id]/+page.server.js load`
  → `[id]/+page.svelte` breadcrumb + WO page banner
- NAV confirmed broken instances: `/po/[id]` Back, `/po/new` Cancel,
  `/wo/[id]/confirm` Cancel, `/customers/[id]` save
