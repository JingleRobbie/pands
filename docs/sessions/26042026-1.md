## Decisions Made

- **Amber highlight only on past-due dates**: future DRAFT shipments don't trigger the amber alert ‚Äî only `dateStr < todayStr && hasDraft`.
- **"This Week" is rolling, not Mon‚ÄìSun**: starts today‚àí3, ends today+7. Past drafts fetched separately via server `past_drafts=1` param and prepended to results.
- **Business day / holiday filtering is implicit**: the calendar only renders dates that have shipments, so non-business-day empty days never appear. No explicit filter needed.
- **group_id fallback**: `run.group_id ?? run.id` ensures runs that predate the groups table (or have no group) still render correctly as individual rows.
- **Facing displayed as joined string**: "OB, IB" rather than dropping or picking first ‚Äî preserves all information in the facing column.
