## Decisions Made

- **Removed POs → CANCEL not DELETE:** Preserves history; consistent with
  existing cancel action on PO detail page.
- **Removed POs default unchecked** in diff approval — safe default, user
  must opt in to cancellations.
- **No sticky columns:** Two-row colspan/rowspan header makes column freezing
  unreliable; SKU filter buttons provide sufficient narrowing instead.
- **Sticky header rows tried and reverted:** User decided against it this
  session.
- **10px font via direct CSS:** Tailwind has no 10px preset; `font-size: 10px`
  set directly on `.matrix-table th` and `.matrix-table td`.

## Context Anchors

- Matrix CSS lives in `src/app.css` under `@layer components` — all matrix
  classes (`.matrix-table`, `.row-*`, `.sqft-*`, `.badge` override) are there.
- 10px font is set via raw `font-size: 10px` on `.matrix-table th` and `td`,
  NOT via Tailwind class — don't add `text-xs` inline or it overrides to 12px.
